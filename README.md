# reframe-fsm

A collection of example [reframe]() projects.

## Examples

* login-fsm
* register-no-fsm

## Usage

From the `user` namespace at the REPL:

```clojure
user=> (go) ;; start webserver
...
user=> (watch-cljs) ;; start cljsbuild auto
...
```
